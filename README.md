## 🚀 Featured Projects

### 🌬️ [Air Quality Monitoring IoT](https://github.com/Bio-Tingi/Air-Quality-Monitoring-IoT)
> *Master's Research Project — University of Vaasa*

IoT-based indoor air quality monitoring system using **Raspberry Pi Pico W** and **BME680** sensor. Measures temperature, humidity, pressure, and gas resistance to calculate real-time AQI (Good / Average / Poor / Bad). Syncs data to **Firebase** cloud and hosts a local HTTP server for remote monitoring.

`Python` `Raspberry Pi Pico W` `BME680` `Firebase` `IoT` `HTTP Server`

---

### 💊 [Pharmic](https://github.com/Lchamodya/pharmic)
> *Bachelor's Final Year Project — University of Plymouth*

A pharmacy management web application with full CRUD operations for managing pharmaceutical inventory and prescriptions. Designed with an intuitive UI and built end-to-end from requirements to delivery.

`HTML` `CSS` `JavaScript` `PHP` `UI/UX Design`
